About the Mindful Moments Tracker

This application was created for my INF601 Final Project. The goal was to take the mini project from earlier in the semester and build it into something more complete, personal, and functional while demonstrating the skills I learned throughout the course.

The Mindful Moments Tracker is designed to be a calm and supportive space for self reflection. It helps users slow down, look inward, and understand their emotional patterns over time. The design stays soft and simple so the user can focus on how they feel rather than the complexity of the interface.

What You Can Do Here

  • Write daily reflections and keep them organized
  • Choose the mood that matches your day
  • See a quote selected to match your emotional energy
  • View your mood history through a calm, easy to read chart
  • Notice patterns in how you feel over time
  • Use a clean, secure login system to keep everything private

Technical Overview

This project uses Flask as the main framework. It includes a full folder structure with blueprints for clean separation of pages and authentication. A SQLite database stores users, reflections, and moods. The ZenQuotes API provides mood related quotes, and Matplotlib is used to generate the visual mood chart on the dashboard.

These features demonstrate database management, user authentication, API integration, visualization techniques, and modular application design which were all key components of the course.
